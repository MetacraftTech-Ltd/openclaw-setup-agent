## What This Software Does

OpenClaw Setup Agent is a CLI tool that helps you configure [OpenClaw (Clawdbot)](https://github.com/clawdbot/clawdbot), an AI assistant framework. It:

- Analyzes your system environment
- Guides you through configuration
- Generates a `clawdbot.yaml` file
- Validates your setup
